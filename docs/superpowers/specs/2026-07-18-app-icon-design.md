# App Icon Design — MF-Mathematics

**日期**: 2026-07-18
**状态**: 已完成

---

## 需求概述

为 Multifunctional-Mathematics 数学软件设计极简应用图标：
- 主体：波浪曲线 + 几何立方体组合成字母 "M"，象征数学分析与多维可视化
- 配色：深蓝 `#1a365d` + 亮青 `#00d4ff`，白色背景
- 风格：扁平化矢量，方形构图，无文字（图标部分），高对比度

## 设计方案：「曲直共生」

**M 结构**基于等距投影立方体的顶点：

| 顶点 | 坐标 | 角色 |
|------|------|------|
| T | (256, 55) | M 中心峰 |
| TL | (117, 135) | M 左上角 |
| TR | (395, 135) | M 右上角 |
| BL | (117, 295) | M 左下脚 |
| BR | (395, 295) | M 右下脚 |
| F | (256, 215) | 立方体前角点 |

**M 四笔**：
1. **左竖笔** — 立方体棱线 TL→BL，深蓝色粗杆
2. **左斜笔** — 正弦波曲线 BL→T（1.5 周期），亮青色
3. **右斜笔** — 直线 T→BR，深色强调
4. **右竖笔** — 立方体棱线 BR→TR，深蓝色粗杆

**立方体三个可见面**（等距投影）：
- 顶面 T-TL-F-TR（最亮）
- 左面 T-F-BL-TL（中等）
- 右面 T-TR-BR-F（最暗）

## 文字布局

图标下方居中排列：
- 上行 `MULTIFUNCTIONAL`（轻字重 300，27px，字距 3.5）
- 亮青色分隔线
- 下行 `MATHEMATICS`（粗字重 600，31px，字距 5.5）

字体栈：Segoe UI → Helvetica Neue → Arial → sans-serif

## 输出

- SVG 矢量文件：`assets/icon.svg`（512×640 viewBox）
- 可导出为任意尺寸 PNG/ICO

## 验证方法

1. 在浏览器中打开 `assets/icon.svg` 预览
2. 缩放到 64×64 / 256×256 确认小尺寸可辨识
3. 使用 ImageMagick 或在线工具转换为 ICO 格式
