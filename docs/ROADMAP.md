## 一、项目使命

打造**开源数学桌面软件标杆** — 覆盖纯数学到应用数学全分支，集成 AI 智能辅助，兼顾专业深度与易用性。

---

## 二、五大战略方向

### 🎨 一、绘图引擎

**愿景**：5 种模式全部达到专业水准，渲染性能桌面级（缩放 60fps）。

| 模式 | 当前 | 目标 |
|------|:--:|------|
| 2D 函数 | ✅ 成熟 | 极坐标、参数曲面、动画 |
| 3D 曲面 | 🔶 基础 | 旋转/光照/等高线/颜色映射/动画 |
| 复数着色 | 🔶 基础 | HSV 域着色、网格映射、零点极点标注、Riemann 球面 |
| 向量场 | 🔶 基础 | 流线动画、散度旋度热力图、粒子追踪 |
| 任意做图 | 🔶 基础 | 文字标注、角度/距离测量、SVG/PNG 导出 |

**长期**：脚本动画系统（`plot.animate(expr, t=0..10)`），可导出 GIF/MP4。

---

### 🤖 二、AI 智能化

**愿景**：从"计算工具"进化到"数学导师"，四级能力金字塔。

```
L4 导师   ← 自适应教学 · 知识追踪 · 苏格拉底对话
L3 解题器  ← Agent 推理链 · 自动策略 · 双模型验证
L2 理解器  ← RAG 知识库 · 手写识别 · 语义搜索
L1 基础   ← 对话 · 步骤 · 加速（✅ 已达成）
```

**关键里程碑**：
- L2：嵌入 100+ 经典教材 → 本地向量库，检索增强生成
- L3：Function Calling → AI 可自主调用 sympy 工具链
- L4：用户画像 → 自适应难度 → 个性化习题推荐

**长期**：多 Agent 协作（一个出题、一个解题、一个评判）。

---

### 📐 三、数学深度

**愿景**：每个分支提供"理论 + 数值 + 可视化"三合一体验。

**微分方程**：ODE 方向场/相图/分岔图 → PDE 热方程/波动方程/有限差分可视化

**优化运筹**：线性规划（单纯形）→ 非线性（梯度下降/牛顿法）→ 约束优化（拉格朗日乘数/KKT）→ 凸优化

**统计增强**：ANOVA → 非参数检验 → 时间序列（ARIMA）→ 贝叶斯推断

**几何**：3D 几何体构造 → 微分几何（曲率/测地线/高斯曲率）→ 计算几何

**离散数学**：图论（最短路径/最小生成树/网络流）→ 组合优化 → 密码学可视化

**长期**：范畴论基础、代数几何入门、量子计算数学。

## 三、三阶段路线图

### 近期（1-2 月）：功能扩展
- 5 种绘图模式完善
- AI Agent 推理链 + 题目理解
- 新数学模块（优化/ANOVA）
- 暗色补全 + 撤销/保存

### 中期（3-6 月）：平台化
- AI L2（RAG + 多模态 + 语义搜索）
- 微分几何 + 图论 + 离散数学
- 测试 15% + except 细化
- PyInstaller 打包 + 自动更新

### 远期（6-12 月）：生态化
- AI L3-L4（自适应教学）
- 插件系统 + 在线公式库
- 多语言国际化
- 社区建设

## 四、不做的方向（明确边界）

- ❌ 云端 SaaS（保持桌面应用定位）
- ❌ 实时协作编辑（复杂度远超价值）
- ❌ 移动端（保持桌面专注）
- ❌ 自有 AI 模型训练（使用 API，不造轮子）
- ❌ 3D 游戏引擎级渲染（使用 matplotlib，不自研）
